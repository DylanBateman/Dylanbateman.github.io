    Bay Fisher

    Unity, C#


    Bay fisher is a idle fishing game project that was created during my final year of university in my mobile games development module. This module involved coming up with a unique idea for a game and taking it from the designing stage, through development and into testing and optimisation ready to be released.


    The project began with creating a detailed game design document covering gameplay related ideas such as mechanics, art style, UI, and assets aswell as interaction and screen designs. Strategies for monetisation and user retention were also considered to consider logentivity and success for the game within the competitive mobile market.


    The game then moved into the development stage and then once developed the Unity profiler was used to optimise the game to ensure it would run well on the limited resources avaliable on mobile devices.


    Overall, this was an enjoyable project where I got to use my skills developed during my time at university to develop a game as well as learn new skills and give new considerations to ideas and features not considered in previous projects.
